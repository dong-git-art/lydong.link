搭建个人博客步骤简单汇总

准备工作

  • 安装R/Rtudio/Git/Hugo
  • 阿里云/腾讯云购买自己心仪的域名并解析，云服务器
  • git简单命令（建库与托管）

开始

1.安装blogdown、hugo,打开Rstudio，键入命令：

install.packages(“blogdown”)

blogdown::install_hugo()

2.新建一个Rproject

3.<Hugo主题页选择主题，复制githubUserName/githubRepository > https://themes.gohugo.io/

4.修改网页内容，运行blogdown::serve_site()本地预览

5.github创建仓库，设置.ignore文件

6.git进行上传、同步

  • 绑定github-ssh

cd 项目目录

git init

git add .

git commit -m "备注“

git push -u origin master(master为分支名)

7.netlify进行github部署(build)

8.域名解析——阿里云

9.SSL证书

10.等待一定时间后可以进行访问(可能由于网络出口原因，有时必须科学上网)
